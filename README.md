# GeoMPNN reproduction

Re-implementation of [**GeoMPNN**](https://arxiv.org/abs/2412.09399) (Helwig et al.,
NeurIPS 2024 ML4CFD competition — Best Student Submission, 4th overall) on the
**AirfRANS** dataset under the **2024 ML4CFD competition** scoring.

## Layout

```
src/geompnn/        # model + data loader + scoring + train / eval entry points
src/geompnn/configs/  # 10 ablation configs (mlp, gnn, s2v, s2v_gnn, trail, polar, sine, sph, inlet, geompnn)
scripts/setup.sh    # one-shot env build + AirfRANS download
scripts/run_full_sweep.sh  # turn-key driver for the 80-run sweep
scripts/sweep.py    # orchestrator (handles MIG UUIDs and rolling ETA)
scripts/summarize.py # collect score.json files → results/summary.md
data/airfrans/      # AirfRANS raw simulations (~17 GB; downloaded by setup.sh)
results/<ab>/seed<k>/  # per-run outputs (parameters, ckpt, eval.json, score.json)
logs/               # tee'd sweep logs
env/                # Python 3.10 venv (matches paper's torch 2.0.1+cu118 stack)
```

The scratch/notes directory `llmdocs/` is .gitignored — it's the coding
agent's external brain and is not part of the deliverable.

## Quick start (new machine)

```bash
bash scripts/setup.sh           # ~25 min: build venv, install deps, download AirfRANS (~10 GB compressed)
bash scripts/run_full_sweep.sh  # the full 80-run reproduction
```

On **Compute Canada / DRAC** clusters (e.g. `*.computecanada.ca`,
`platogpu*`), `setup.sh` auto-delegates to `scripts/setup_cc.sh`, which uses
the CVMFS wheelhouse for torch + PyG (closest available is torch 2.1.1, a
one-minor drift from paper's 2.0.1 — within reproduction tolerance) and
reaches PyPI via `--index-url` only for `lips-benchmark` + `airfrans`. Make
sure the right modules are loaded first:

```bash
module load python/3.10 cuda/11.8
bash scripts/setup.sh           # delegates to setup_cc.sh on CVMFS hosts
```

`scripts/run_full_sweep.sh` auto-detects MIG instances via `scripts/detect_gpus.sh`
and runs one parallel training per slot. Override slot detection by setting
`GEOMPNN_GPUS="MIG-uuid1 MIG-uuid2 ..."` in the environment.

The script prints a `FINAL ETA` line at startup, on every per-run completion,
and every 5 minutes — initially using a 9 h/run placeholder (our measured A40
baseline), then refined to the rolling mean once runs complete.

### Subset / smoke runs

```bash
# Smoke (2 epochs, 4 sims, no eval) — verifies the whole pipeline in ~2 min/slot
bash scripts/run_full_sweep.sh --ablations s2v --seeds 0 \
    --override nb_epochs=2 subsampling=1500 --limit_train 4 --skip_eval --force

# Just GeoMPNN-final, all 8 seeds
bash scripts/run_full_sweep.sh --ablations geompnn --seeds 0 1 2 3 4 5 6 7
```

The sweep is **resumable**: re-running skips any (ablation, seed) that already
has `results/<ab>/seed<k>/score.json`.

## Environment

- Python 3.10
- torch 2.0.1+cu118, PyG 2.7.0, lips-benchmark[recommended]==0.2.7, airfrans 0.1.5.1
- Tested on 2× NVIDIA A40 (CUDA 12.8 driver) and on 2× NVIDIA RTX PRO 6000
  Black w/ MIG (CUDA 13.1 driver). cu118 binaries are forward-compatible.
- `requirements.txt` lists the curated top-level pins; `requirements.lock.txt`
  is the full pip-freeze snapshot for byte-exact reproducibility.

## Reproduction targets (from paper §4)

| Ablation       | Target Global Score (median over 8 seeds) |
|----------------|------------------------------------------:|
| MLP            | ~22                                       |
| GNN            | ~28                                       |
| S2V            | ~36                                       |
| TRAIL          | ~43                                       |
| POLAR          | ~45                                       |
| SINE           | ~44                                       |
| SpH            | ~47                                       |
| INLET          | ~52                                       |
| **GeoMPNN**    | **~53–55** (must beat Casenave 47.04)     |

Plus the negative-result ablation: **S2V+GNN ≈ S2V** on every field.

`results/summary.md` is regenerated by `run_full_sweep.sh` after the sweep
completes; it tables reproduced medians, paper targets, deltas, and pass/fail
at ±2 score-point tolerance.
