# GeoMPNN reproduction — top-level dependency stack

# Install order matters: torch (cu118 wheels) must land before LIPS so that
# the [recommended] extra's `torch==2.0.1` pin is satisfied without re-fetch,
# and PyG geometric extensions need to match torch 2.0.1+cu118 exactly.
#
# Use scripts/setup.sh — it does this in the right order.
#
# Manual:
#   pip install --upgrade pip wheel setuptools
#   pip install --index-url https://download.pytorch.org/whl/cu118 \
#     torch==2.0.1 torchvision==0.15.2 torchaudio==2.0.2
#   pip install -f https://data.pyg.org/whl/torch-2.0.1+cu118.html -r requirements.txt
#
# Tested on Python 3.10.12. Works on driver >= 520 (cu118 binaries are forward-
# compatible with newer drivers, including CUDA 13.x).
